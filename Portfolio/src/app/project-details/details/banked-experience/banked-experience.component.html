Banked Experience

This project is a Java plugin for the RuneLite client, which is an open-sourced game client made for Old School RuneScape. This plugin automates the calculation of experience a player would received based on the items they have accumulated in their bank. This is accomplished by reading the contents of the players bank and displaying the results within a user interface written with the Java Swing library. This UI also provides a way for the player to customize the functionality of the plugin, such as ignoring specific items.
