Mello Trainer!

Mello Trainer is a FiveM server-side trainer written in Lua with a UI written using HTML5, CSS, and JavaScript. A trainer is a generic tool used to do just about everything inside the GTA 5 world and are highly utilized inside the role-playing community.

Features

  • Admin Menu
  • Customizable access levels
  • Dynamic menu creation based on JSON
  • Online Players Menu w/ Spectate & Teleport Option
  • Player modifier Options (God Mode, Infinite stamina, etc)
  • Player Join/Leave Notifications
  • Player Death Messages
  • Player Skin Changing and Customization
  • Vehicle Spawning, Customization, and Modifications
  • Customizable Vehicle and Skin presets for players
  • Weapon Spawning & Attachments
  • Voice Overlay & Options
  • General settings (Hide Hud, Hide Map, Map blips)
  • No-clip abilities with admin-only toggle
  • Version check on startup
