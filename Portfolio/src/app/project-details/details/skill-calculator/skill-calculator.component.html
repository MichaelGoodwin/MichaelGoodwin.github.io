Skill Calculator Expansion

For this project my goal was to expand upon the existing RuneLite "SkillCalculator" plugin by adding Planner and Banked Experience functionality. Eventually this was decided against as this would require a lot of maintenance and there are websites that offer similar tools.

View the source code

Banked Experience

The main purpose of this project was to add a Banked Experience calculator to the client. To accomplish this i needed to read the contents of a players bank and search for items used to train specific skills. Certain items can be used in multiple activities which reward different experience values so i also needed to allow the player to select which activity they are doing, defaulting to first one. Some items are also turned into other items which means this had to be recursive friendly.

Planner

In order to add the Planner functionality i copied the existing calculator UI and added a right-click menu for specifying the quantity. The planner won't let the player select activities they don't have the level for. The existing target level and target experience boxes are updated to reflect these changes.
