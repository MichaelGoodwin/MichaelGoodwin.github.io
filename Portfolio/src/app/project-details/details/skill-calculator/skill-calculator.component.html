Skill Calculator Expansion

This is a expansion to the existing Skill Calculator plugin inside the RuneLite client, which is made for Old School RuneScape. This expansion added two new optional tabs to the existing Swing UI: Planner and Banked Exp. Eventually this was decided against as this would require a lot of maintenance, there are websites that offer similar tools, and some of the logic was overly complex.

View the source code

Banked Experience

The purpose of this tab was to automate the calculation of experience the user would receive from the items they had accumulated in their bank. This calculation has to account for items being used in multiple activities which reward different experience values and items turning into other items which reward experience while also being customizable for the end-user. These complications, and many other small edge cases, eventually ended up with some overly complex code and was decided against as there are websites which already have this functionality.

Planner

The planner tab is a copy of the existing UI with an added a right-click menu for specifying the quantity. The calculation for this is a simple loop which runs every time a change is made to the planner quantities. The existing target level and target experience boxes are also updated to reflect these changes based on this calculation.
