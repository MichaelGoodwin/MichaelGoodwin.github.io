Loot Logger

This project is a Java Plugin for the RuneLite client, which is an open-sourced game client made for Old School RuneScape. This plugin logs the items, called loot, received from killing monsters or completing specific activities within the game to the player's local device. The data is stored utilizing the JSON Lines format, appending new entries to the existing file(s). The plugin also provides an optional user interface for viewing the stored data that was written using Java's Swing library.

This plugin works in conjunction with the official Loot Tracker plugin, which can be found here.
