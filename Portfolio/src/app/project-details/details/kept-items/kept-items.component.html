Kept Items

For this project my goal was to create a plugin that reworked the existing "Items Kept on Death" interface for Old School RuneScape. This was possible using the open sourced third-party client, RuneLite. The company behind the game has since announced a rework to this entire system so the plugin was put on hold until we know if it's worth adding.

View the source code

Purpose

The game servers for Old School RuneScape were experiencing targeted ddos attacks to ensure players would die and lose their items. To combat this issue the developers implemented a temporary death mechanic which has since been in place for over 3 years but they have not updated the in-game interfaces to reflect these changes. This plugin was meant to update this interface to more accurately display what will happen when your player dies. The interface was also updated to be interactive as there are different factors affecting items kept and the default interface looks at just the current factors affecting the player.
