Kept Items

This is a RuneLite plugin that reworks the existing "Items Kept on Death" interface for Old School RuneScape. The game servers were experiencing targeted ddos attacks which resulted in the game developers implementing "temporary" death mechanics, which have since been in place for over 3 years. This plugin is written in Java and is an open-sourced contribution that was merged on Jun 18th, 2019.
