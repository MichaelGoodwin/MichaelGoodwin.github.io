Loot Tracker

For this project my goal was to create a plugin that tracked loot received from specific monsters and activities for the game Old School RuneScape using the RuneLite client. Although my version was not implemented i helped implement the current version available in the default client, which was announced in this blog post. I also helped create the API used for the main client as the maintainers did not want to serialize/deserialize local data which i was doing by saving the loot into `.log` files.

View the source code

Boss Logger

The original name for this plugin was "Boss Logger" as it only supported specific monsters, which were considered by many to be the bosses of the game. This version started with a very rough UI and no layout customization and eventually morphed into the current UI design. Once we decided on including all in-game monsters and other players the name was changed to "Loot Tracker". You can view the Original PR i created with this name here.

Differences

The official Loot Tracker plugin can be viewed here, with the main difference being the UI. The functionality behind determining the loot received is identical but mine will also store data on local files and read from these files when displaying loot. I also added a replay system which cycles through the loot.
