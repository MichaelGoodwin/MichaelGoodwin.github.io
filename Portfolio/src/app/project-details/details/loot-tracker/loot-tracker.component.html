Loot Tracker

This is a RuneLite plugin for Old School RuneScape that tracks the items (loot) received from killing monsters or completing specific activities within the game. Although my version was not implemented i helped implement the current version which was announced in this blog post including the API endpoints used for data persistence between client sessions. This plugin is written in Java using Swing for the UI and includes optional local data storage.

Boss Logger

The original name for this plugin was "Boss Logger" as it only supported specific monsters, which were considered by many to be the bosses of the game. This version started with a very rough UI and no layout customization but eventually morphed into the current UI design. Once the descision was made to including all in-game monsters and other players the name was changed to "Loot Tracker". You can view the Original PR i created with this name here.

Differences

The official Loot Tracker plugin can be viewed here, with the main difference being the UI and optional local data storage capabilities. The functionality behind determining the loot received is identical but the UI they went with was created by another developer. The UI i created included reading from the local data files and a replay system which cycles through the loot for a specific activity in chronological order.
